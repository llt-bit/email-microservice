## 当前问题诊断

### 1. 响应格式问题（最严重）

OA 前端经过 axios 拦截器 `return response.data` 后，`res` = 后端 JSON body 本身。

OA 返回格式（`JsonResult`）：
```json
{"code":"00010001","msg":{"records":[...],"total":N}}
```

前端代码期望 `res.code == 200` 或 `res.code == "00010001"`，然后用 `res.msg` 或 `res.data`。

**目前问题**：counts、list 已改成 OA 格式，但 store 里有兼容性问题。
